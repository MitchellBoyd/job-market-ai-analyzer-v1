# AI Job Market Analyzer

An AI-powered data analysis project that processes job market data, generates insights, and produces automated reports.

---

## 📊 Features

- Clean and preprocess job dataset (50,000+ records)
- Analyze job demand, salaries, and locations
- Visualize trends using charts
- Generate AI-powered insights using OpenAI
- Automatically save analysis reports

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- OpenAI API

---

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run data analysis:
   python main.py

3. Run AI insights:
   python ai_analysis.py
